Tecnologiás móviles en sitios web

Disponer de una página web optimizada para dispositivos móviles siempre ha supuesto una ventaja. El diseño adaptado para móviles es ahora esencial para el desarrollo de una web.

  • Ver más

Principales Tecnologías

  • HTML 5

    HTML5 es la última versión de HTML. El término representa dos conceptos diferentes. Se trata de una nueva versión de HTML, con nuevos elementos, atributos y comportamientos.

  • ViewPort

    El viewport del navegador es el área de la ventana en donde el contenido web está visible. Para todo tipo de dispositivos móviles.

  • CSS

    CSS, es una tecnología que nos permite crear páginas web de una manera más exacta. Gracias a las CSS somos mucho más dueños de los resultados finales de la página, pudiendo hacer muchas cosas que no se podía hacer utilizando solamente HTML, como incluir márgenes, tipos de letra, fondos, colores...

Características

  • HTML 5

    Diseñado para ser utilizable por todos los desarrolladores de Open Web, esta página referencía numerosos recursos sobre las tecnologías de HTML5, clasificados en varios grupos según su función.

  • ViewPort

    Como definición rápida, diremos que viewport podría traducirse como vista o ventana y nos sirve para definir qué área de pantalla está disponible al renderizar un documento, nivel de escalado y el zoom que debe mostrar inicialmente. Todo ello, con parámetros que le damos a la propia etiqueta META.

  • CSS

    El modo de funcionamiento de las CSS consiste en definir, mediante una sintaxis especial, la forma de presentación que le aplicaremos a los elementos de la página.

Ventajas

  • HTML 5

    Ventajas que ofrece CSS: Incorporación de video. Experiencia de usuario. Un código más limpio. Compatibilidad múltiple entre navegadores. Optimización para móviles. Una mejor posibilidad de ranking.

  • ViewPort

    Las ventajas de usar el ViewPort son: El viewport virtual es una forma de resolver el problema de sitios no optimizados para móviles, logrando que se vean mejor. Es un estándar que ya está soportado por la mayoría de dispositivos.

  • CSS

    Ventajas que ofrece CSS: Podemos definir la distancia entre líneas del documento. Se puede aplicar identado (sangrado) a las primeras líneas del párrafo. Podemos colocar elementos en la página con mayor precisión, y sin lugar a errores. Y mucho más, como definir la visibilidad de los elementos, margenes, subrayados, tachados...
